# Decision Fatigue Killer

Eliminate decision fatigue by removing choice entirely.

## Usage

1.  **Input**: Select your available Time, Energy level, and current Mood.
2.  **Execute**: The system assigns ONE task. You are locked in.
3.  **Complete**: Do the task. Mark as done.

## Tech Stack

-   Next.js (App Router)
-   TypeScript
-   Minimal CSS Variables (Dark Theme)

## Development

```bash
npm install
npm run dev
```

Built for serious work.
